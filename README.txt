PROYECTO: SITIO WEB PLAYSTATION
================================

DESCRIPCIÓN:
Sitio web responsive de PlayStation con sistema de autenticación,
diseño moderno y funcionalidades interactivas.

CARACTERÍSTICAS PRINCIPALES:
✅ Diseño responsive estilo PlayStation
✅ Sistema de registro y login de usuarios
✅ Persistencia de sesión con localStorage
✅ Navegación dinámica según estado de sesión
✅ Notificaciones visuales
✅ Carrusel interactivo
✅ Secciones de juegos destacados y noticias

ESTRUCTURA:
- inicio.html: Página principal con carrusel y contenido
- login.html: Formulario de inicio de sesión
- registro.html: Formulario de registro de usuarios

TECNOLOGÍAS UTILIZADAS:
- HTML5, CSS3, JavaScript
- Bootstrap 5.3.0
- Bootstrap Icons
- LocalStorage para persistencia

DEPENDENCIAS DEL PROYECTO:
- Bootstrap 5.3.0
- Bootstrap Icons 1.11.0

ARCHIVOS DE CONFIGURACIÓN INCLUIDOS:
- package.json (configuración de npm)
- .gitignore (archivos a ignorar en control de versiones)

================================================================
INSTRUCCIONES DE INSTALACIÓN - MODO SIMPLE (RECOMENDADO)
================================================================

MÉTODO 1: EJECUCIÓN DIRECTA (Sin npm)
--------------------------------------
1. Descomprimir el archivo ZIP
2. Abrir inicio.html en cualquier navegador web
3. No requiere servidor - funciona localmente
4. Todas las dependencias están incluidas via CDN

MÉTODO 2: INSTALACIÓN CON NPM (Para desarrollo)
-----------------------------------------------

PREREQUISITOS:
- Node.js instalado (versión 14 o superior)
- npm (viene incluido con Node.js)

PASOS DE INSTALACIÓN:

1. Descomprimir el proyecto
2. Abrir terminal en la carpeta del proyecto
3. Instalar dependencias:
   ```bash
   npm install
